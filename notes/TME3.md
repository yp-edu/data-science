# Dimensionality Reduction

Very important to visualize high-dimensional data, but also to process the date before using a machine learning model.

- Selection vs features creation:
    - Creation is often easier
    - Selection is better for interpretable models
- PCA: can be used for visualization or preprocessing
- t-SNE: only for visualization
